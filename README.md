# horror_game
오부 쯔꾸르 게임 WinAPI 제작

[ 기획안 내용(리드미에 임시저장) ]

낮

1. 의뢰인 npc 찾으러 마을 돌아다니기
2. 의뢰인 찾고 말걸기, 말 건 후 오늘 밤에 묵을 곳 안내받기(묵을 곳에서 세이브 저장)
->의뢰인이 예전에도 퇴마사를 부른 적이 있는데 도망갔다고 하기
3. 마을 주민 npc들을 통해 곳곳에 숨어있는 색깔별 부적 찾기
4. 부적 다 찾으면 마을 npc나 마을 동물들 or 집에 버려진 일기장들이나 쪽지들을 통해 밤마다 어디에서 불가사의한 현상이 일어나는지 장소 알아내기
->의뢰인은 봉인해야할 장소를 5개라고 말하지만 일기장이나 쪽지에는 6개라고 적혀있음 
->마지막 봉인은 살아있는 그릇이어야 한다. 라는 쪽지나 뭐 아무튼 일기장이나 쪽지에는 반전요소 떡밥or 힌트 던져놓기
5. 집마다 결계 치기

밤 -> 요괴 설치

1. 마을 주민들과 같이 모여 저녁 식사하기
2. 집에 가서 세이브 한 후 본격적으로 봉인 시작하기
3. 낮에 찾은 장소에 가서 요괴를 피해 부적 붙이고 봉인 시키기(이런 방식으로 5개 장소 각각 봉인시키러 다니기)
4. 다 봉인 시키고 나서 집에 돌아가기

다음날 아침
1. 의뢰인 npc한테 찾아가서 봉인 다했다고 말하기
2. (여기서부터 그냥 스토리형식, 플레이 x) 의뢰인이 사실 숨겨진 봉인 하나가 남아있다고 말한 후 주인공이 뭐냐고 물어보면 너라고 대답하면서 화면 빨간색으로 바뀐 후 웃는 소리 나온 후 끝내기

요괴 종류(변경 가능)
-구미호 > 이동 속도 주인공의 +3 빠름, 불꽃 뿜음 따로 피할 방법 x(잡히거나 불꽃에 닿을 시 hp -2 감소)
-그림자 귀신->순간이동 가능, 순간이동 후 3초 후에 그 자리에 있는 대상 잡아먹음(hp -3 감소)
-까막 귀신->앞을 볼 수 없지만 소리에 예민( 공격받을 시 hp -1)
-퀴즈 요괴->자신의 퀴즈를 다 맞추기 전까진 절대 못벗어남(퀴즈 수=총 6개) (퀴즈 틀릴 시 1문제당 HP -1) 
-도깨비-> 주인공한테 돌 던짐 (HP -1 감소)
